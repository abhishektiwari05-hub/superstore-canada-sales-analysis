# Superstore Canada Sales Analysis Dashboard

## Project Overview

This project analyzes Superstore Canada sales data using SQL and Power BI.

## Tools Used

- Power BI
- MySQL
- Excel
- DAX
- Power Query

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Order Quantity
- Shipping Cost
- Unit Price
- Profit Margin

## Dashboard Pages

### 1. Executive Dashboard
- Sales and Profit KPIs
- Order Quantity
- Shipping Cost
- Unit Price
- Sales by Region
- Sales by Product Category
- Profit by Customer Segment

### 2. Product Performance Dashboard
- Top 10 Products by Sales
- Bottom 10 Products by Profit
- Sales by Product Sub-Category
- Sales vs Profit by Customer Segment
- Interactive filters for Customer Segment and Product Sub-Category

### 3. Customer Analysis Dashboard

## Key Insights

- Highest Sales Region
- Top Selling Products
- Bottom 10 Products by Profit
- Customer Segment Performance
- Product Category Analysis
- Sales Trend Analysis
