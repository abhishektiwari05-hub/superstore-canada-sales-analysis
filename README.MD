# Superstore Canada Sales Analysis Dashboard

## Project Overview

This project analyzes Superstore Canada sales data using SQL and Power BI.

## Tools Used

- Power BI
- MySQL
- Excel
- DAX
- Power Query

## KPIs

- Total Sales
- Total Profit
- Total Orders
- Profit Margin

## Dashboard Pages

1. Executive Dashboard
2. Product Performance
3. Customer Analysis
4. Profit Analysis

## Key Insights

- Highest Sales Region
- Top Selling Products
- Loss Making Products
- Customer Segment Performance
